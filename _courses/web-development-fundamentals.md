Welcome to Web Development Fundamentals! This course will teach you everything you need to know to start building websites from scratch.

## What You'll Learn

- Structure web pages with HTML
- Style your pages with CSS
- Add interactivity with JavaScript
- Best practices for modern web development

## Prerequisites

No prior programming experience required. Just bring your curiosity and a computer!

## Course Format

Each lesson includes text explanations, code examples, and practical exercises. Work through the modules in order for the best learning experience.
